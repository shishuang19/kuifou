# 产品需求文档 (PRD)
## 亏否 - 个人资产折旧管理应用

---

## 1. 产品概述

### 1.1 产品定位
亏否是一款个人资产折旧管理应用，帮助用户记录、追踪和管理其购买的物品，自动计算折旧成本，让用户清晰了解每件物品的实际价值和每日成本。

### 1.2 目标用户
- 个人用户（仅个人使用）
- 关注资产管理的消费者
- 希望了解物品使用成本的理性消费者

### 1.3 核心价值
- 记录每一件心爱之物
- 自动计算折旧和每日成本
- 可视化资产统计
- 帮助用户理性消费决策

## 3. 数据计算逻辑

### 3.1 折旧计算公式

```
购买价格 = 商品购买时的价格
使用天数 = 当前日期 - 购买日期
每日成本 = 购买价格 / 使用天数
当前残值 = 购买价格 - (每日成本 × 使用天数)
```

### 3.2 统计计算

```
总残值 = Σ 所有资产的当前残值
今日成本 = Σ 所有资产的每日成本
资产数量 = 物品总数
预计每日成本 = 今日成本（实时更新）
```

- 页面切换流畅，无明显卡顿

- 注销账号时清除所有数据

### 4.3 用户体验
- 深色主题为主
- 紫色渐变风格
- 简洁直观的界面设计
- 符合移动端操作习惯

### 4.4 兼容性
- iOS 13.0+
- Android 8.0+
- 响应式布局适配不同屏幕尺寸

## 5. UI/UX 设计规范

### 5.1 颜色方案
- 主色调：紫色渐变 (#6366F1 → #8B5CF6)
- 背景色：深色主题 (#1A1A2E, #16213E)
- 卡片背景：深蓝灰色 (#0F2027, #203A43)
- 文字：白色 (#FFFFFF)
- 次要文字：灰色 (#9CA3AF)
- 强调色：天蓝色用于天数显示

### 5.2 排版
- 主标题：24-28pt
- 卡片标题：18-20pt
- 正文：14-16pt
- 辅助信息：12pt

### 5.3 图标设计
- 使用emoji或自定义图标
- 圆角正方形容器
- 统一尺寸 48x48pt

### 5.4 卡片设计
- 圆角半径：16pt
- 内边距：16-20pt
- 阴影：轻微投影增强层次感
- 渐变背景增强视觉效果
